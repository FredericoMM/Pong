# 
Mobile Pong game

## Features
- Two game modes 
  - One player mode
  - Two players mode
- Three difficulty levels
  - Easy
  - Medium
  - Hard
- Two types of paddles
  - Simple (Easy)
  - Smooth (Medium and Hard)
